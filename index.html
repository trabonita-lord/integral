1) What is “net area”?

When we say net area (also called signed area), we mean: area above the x-axis counts as positive, while area below the x-axis counts as negative.

Why do we need “signed” area?

A definite integral ∫ f(x) dx measures accumulation. If f(x) is positive, the accumulation increases; if f(x) is negative, the accumulation decreases. That behavior matches “positive area” and “negative area”.

2) The definite integral as net area

For a continuous function f on [a,b], the definite integral ∫ab f(x) dx equals the net area between the graph of f and the x-axis from x=a to x=b.

Above the x-axis

If f(x) ≥ 0 on [a,b], then ∫ab f(x) dx equals the usual geometric area.

Below the x-axis

If f(x) ≤ 0 on [a,b], then ∫ab f(x) dx is negative and equals minus the geometric area.

Important: net area vs total area

The integral gives net area. If you want total area (always positive), you usually compute ∫ab |f(x)| dx or split into pieces and add geometric areas.

3) How to compute net area from a graph (practical method)

  1. Mark the interval on the x-axis: from a to b.
  2. Find x-intercepts of the function (where it crosses the x-axis). These are points where the sign of f(x) can change.
  3. Split the integral at each intercept: ∫ab f(x)dx = ∫ac f(x)dx + ∫cb f(x)dx (and continue if there are more intercepts).
  4. Compute each piece. Pieces above the axis contribute positive values; pieces below contribute negative values.
  5. Add them to get the net (signed) total.

4) Worked examples (net area)

Example A: One region above the axis

Compute ∫02 x dx. The graph y = x is above the x-axis on [0,2].

∫02 x dx = [x²/2]02 = (4/2) - 0 = 2

Interpretation: net area = geometric area = 2.

Example B: One region below the axis

Compute ∫02 (-x) dx. The graph y = -x is below the x-axis on [0,2].

∫02 (-x) dx = [-x²/2]02 = -(4/2) - 0 = -2

Interpretation: net area is negative because the region is below the axis.

Example C: Crosses the axis (split into pieces)

Compute ∫-12 (x-1) dx. The function f(x)=x-1 crosses the x-axis at x=1.

∫-12 (x-1) dx = ∫-11 (x-1) dx + ∫12 (x-1) dx
= [x²/2 - x]-11 + [x²/2 - x]12
= ((1/2 - 1) - (1/2 + 1)) + ((2 - 2) - (1/2 - 1))
= (-1/2 - 3/2) + (0 - (-1/2)) = (-2) + (1/2) = -3/2

Net area is negative overall, meaning “more area” lies below the x-axis than above it.

Quick self-check

  • If the graph is mostly below the x-axis on the interval, the integral should be negative.
  • If the graph is symmetric above and below (equal magnitudes), the net area may be 0.
  • If you want total area, use |f(x)| (absolute value), not f(x).

Summary

  • Definite integrals represent accumulation and equal net (signed) area from a to b.
  • Regions above the x-axis contribute positive area; below contribute negative area.
  • When a graph crosses the x-axis, split the integral at the intercept(s).
  • If you want total area, compute ∫ |f(x)| dx (or split and add magnitudes).
